Start with Basecamp

  1. Create Priorities
  2. Create the Basecamp Todo List and entitle it.
  3. Fill out the description of the list.
  4. Enter single action items with using the @e flag for time estimation.
  5. Add the @s and @m flags to the todo list title.

Example Todo List Title

Audio Tokens @s2013-05-08 @m2013-05-11

Setup

  1. Create a Taskcamp Task.
  2. Make an entry in: Specifications > Summary

Design Files

  1. Upload all comps and design files to the Taskcamp Task.

Estimating Time

  1. Create all Taskcamp Task Todos and estimate the time required for each.
  2. If your estimate is over 3 hours consider breaking into multiple todos.
